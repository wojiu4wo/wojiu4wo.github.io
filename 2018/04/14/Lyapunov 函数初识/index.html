Lyapunov 初体验

  • L优化是通过L函数进行实现的，主要用于control theory。
  • 一个多维state的系统，L函数是这个多维state的scalar。
  • L函数的初衷是： 当系统往undesirable的state前进是，其值会越来越大，且增长速度越来越快，因此可以用来很好的判断方向的正确性，其中控制的手段就是 使 L的drift（L漂移） 为负，也就是往desired state进行塌陷。这个就是所谓的state space collapse.
  • 应用：L drift是queueing network中非常重要的优化control手段。在保证network queues稳定的状态下优化一些performance。这些可以通过在L的漂移中加入weighted penalty term来使得系统往自己想要的方向中发展。

system model

  • network中多个queues，其中每一个queue在t时刻都有某个积压下来的工作, {Q_i}。 比如说我们定义的debt，{d_i for each user}，其中积压的任务就是{d_i - D}。于是我们定义L函数为：
  • $$ L（t）=\frac12 \sum{(Q_i)^2}$$
  • 这个函数是对于整个queues系统中挤压工作的scalar measure。 被称为quadratic lyapunov function on the queue state. 于是我门define
  • $$ \delta (t) = L(t+1)-L(t)$$
  • 这个就是从当前时间点到下一时间点的 差 而已。

the lyapunov drift bound

  • normally, we define ( Q_i(t+1)=max{Q_i(t)+a_i(t)-b_i(t),0} ), 其中这个满足(Q_i(t+1)^2 = max{Q_i(t)+a_i(t)-b_i(t),0}^2 \leq (Q_i(t+1)+a_i(t)-b_i(t))^2)，因为每一次时间的增加，都保证Q不会减小。 通过rearranging 上述不等式，并且sum 且除以2， 最终得到了

$$ \delta(t) \leq B(t) + \sum{Q_i(t)(a_i(t)-b_i(t))} $$

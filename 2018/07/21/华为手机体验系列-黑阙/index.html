黑阙

这个算是非常出名的后台控制的软件了，前身好像是”禁止运行“，广受好评可以把很多进程真正的kill 掉从而达到ios的水平。当然实际效果还带验证

安装

从酷安下载并安装，打开黑阙会提示你如何一步一步设置完成。总而言之：

step1: ADB设置

ADB（android debug bridge）是个用于电脑与android设备进行通信的命令行工具。直接了当去developer.android.google.cn/studio/releases/platform-tools自行下载即可。我用的是mac版本，其实就是下载好之后，按照黑阙的提示得到一个正确的命令行前缀而已。

step2:设置USB调试

需要打开手机的usb调试，这个是对于黑阙需要一定的权限来做到的，因此一般建议不要重启或关机。

step#:输入命令行进行使用

在这里输入命令的时候遇到了bug，其实非常简单

$adb -d shell sh /data/data/me.piebridge.brevent/brevent.sh$

这个是正确命令，注意sh 与 /data之间有空格。
